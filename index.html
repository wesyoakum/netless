Parameter & Method Reference

Mask Detection Parameters

These control which pixels are identified as netting. The mask updates live as you adjust them.

  • Threshold — Brightness cutoff (0–255). Pixels darker than this value are candidates for netting. Raise to catch more of the net; lower if non-net dark areas (jerseys, shadows) are being selected.
  • Morph Size — Size of the morphological opening kernel, as a percentage of the image’s shorter dimension. Opening erodes then dilates the mask — structures thinner than the kernel get removed, large blobs survive. The mask is then subtracted to isolate only the thin structures (the net). Must be larger than the net strand width.
  • Dilate — Expands the mask outward by this percentage of the shorter dimension. Covers the soft, blurry edges of the out-of-focus netting that the threshold alone may miss.
  • Line Filter — Runs a Hough Line Transform on the mask and keeps only pixels near detected line segments. Rejects amorphous dark blobs that aren’t line-shaped (people, equipment, shadows).
  • Connected Only — Keeps only the largest connected region in the mask. The net is one continuous structure, so isolated small blobs (not part of the net) get removed.

Infill Methods

These control how the masked (net) pixels are replaced. Click Process to apply.

  • Diffusion — Iteratively blurs the image and replaces only masked pixels each pass. Surrounding colors gradually flood into net regions. Fastest method. Kernel controls the blur size per pass; Iterations controls how many passes.
  • Telea — Fast Marching Method inpainting. Fills from the mask boundary inward, weighting nearer pixels more heavily. Good balance of speed and quality. Radius controls how far from each masked pixel the algorithm looks for source colors.
  • Navier-Stokes — Fluid dynamics model that propagates both color and gradient direction into masked regions. Preserves edge continuity better than Telea. Similar speed. Uses the same Radius parameter.
  • FFT (Spectral) — Transforms each color channel to the frequency domain, suppresses high-energy periodic peaks (the net’s repeating pattern), and inverse-transforms. Blends the result into masked pixels only. Best for dense, regular netting patterns.

General

  • Scale — Processes at a reduced resolution then upscales. Lower scale = much faster. Use 100% for final output quality.
  • Radius — Inpainting radius as a percentage of the shorter dimension. Larger values fill wider gaps but may soften details.
  • Enable Infill — When off, only the mask is computed and displayed. Useful for dialing in mask parameters without waiting for the infill step.
